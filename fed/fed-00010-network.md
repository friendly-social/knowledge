# Network

> Author: @y9san9, discussed with @demndevel and others
>
> State: Approved

Network is a list of friends, a list of people that both swiped each other right. Actual placement depends on platform and will not be covered there. In the first version there is no pagination, no "important" friends and anything like that. Just a simple list, that backend returns to you. You should be able to view profiles of your friends.

## Future Features

Now Network is sorted by the date when you added a friend, in the future there will be alphabetically sorted list of friends and a list of important people that you often interact with. In the first version we are going to make a big accent in Network. Later on, it's going to be redesigned, so you still can access the list of friends sorted alphabetically and search for them, but the main accent is going to be on QR Codes. For example, combination of [fed-00004-add-friend-qr](fed-00004-add-friend-qr.md) and [fed-00005-blocking-qr](fed-00005-blocking-qr.md).
