# Russia Statement

We try to build the app to improve the world. The problem we are solving is global. Some of the developers are from Russia, some are from Ukraine and other countries. There will be a moment when someone will like to contribute in this project with Russia-specific issue.

Example of such features:

- Builtin VPN to avoid blocks from Russia
- Host an instance in Russsia (or migrate to Russian hosting services)
- Develop and host a relay in Russia
- Add verification or sign-in via gosuslugi

Such kinds of features or requests will not be implemented by this organization. Some developers are still in Russia, but Russia is not our target audience. People from Russia are not banned, but they don't receive any special features as well. If this service is going to be blocked in Russia, no negotiations are going to happen with government and it's better to be blocked in that case.

No Russian infrastructure is used and there are no plans to do so. If any questions because of that will arise to contributors (which all work on this project for free) by government, they are happy to step down and hand this project to developers in other countries.
